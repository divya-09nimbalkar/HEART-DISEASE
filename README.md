# Machine Learning Model Comparison Dashboard

This project is a Flask-based web application that compares the performance of multiple machine learning classification models.

The application displays model accuracy and classification reports, allowing users to analyze and identify the best-performing algorithm.

## Features

- Comparison of multiple machine learning models
- Displays accuracy scores for each model
- Shows detailed classification reports
- Identifies the best-performing model
- Simple web interface using Flask

## Models Used

- Logistic Regression
- Decision Tree
- Random Forest
- KNN (K-Nearest Neighbors)
- SVM (Support Vector Machine)
- XGBoost
- Naive Bayes
- Gradient Boosting
- AdaBoost
- Extra Trees

## Technologies Used

- Python
- Flask
- Scikit-learn
- HTML
